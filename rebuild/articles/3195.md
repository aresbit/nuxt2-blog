# 为什么web3是互联网伊甸园还是数字第三公社？

Web 1.0，也被称为“只读网络”。

互联网早期，Web 1.0的主要目的是在当时通过不同的计算机分享和传播信息。所以人们只能在网页上消费信息，而不能与之互动。

特征是：静态HTML页面，HTTP通信，但某种意义上至今的电子商务也是属于web1，直播电商属于web2.0, 或许web3.0的形式是个人拍卖电商，拍卖各种crypto 形成的基于身份的权利。按下不表，再看看web2.0.

web2.0 被称为“读写”网络或“社交网络”，强调UCG，用户生产内容。用户生产内容意味着用户本身就被异化为商品本身，所以web2.0的算法驱动和后项广告变现导致平台垄断性大公司出现。

网络2.0平台/公司的例子有 Facebook 、谷歌、亚马逊和 Twitter，它们都被称为“科技巨头”。

web2.0特征：

前端交互：javascript ,Vue， React

移动互联

web3开始最初是语义网，意味着计算机能够分析互联网上的任何形式的数据，包括内容、链接和人与计算机之间的交易。

可见人工智能技术和计算机网络组成一台虚拟巨型计算机（即区块链）两者共同构成了web3的核心要义。

在2014年，加文 · 伍兹(Gavin Woods)与人共同创立了“以太网”(Ethereum) ，他创造了“ web3”这个词。

web3是为了解决什么问题呢，从我自己的角度做个比喻，第一公社的共识是交税，第二公社的共识是平台抽成交平台税，第三公社即web3即向不同分布式共识交税，与此同时个人在完成商品异化的同时通过crypto代币确保成为了该组织的股东。

web3创建分布式应用意味着将数据，隐私和信息的权利交给个人出售。

web3特征：进一步交互：VR；安全：rust
